Usage:
	make clean
	make all [START=100, END=1000, NUM_PROCS=16, NB_THREADS=16, SHOW_RESULTS=0]
	sh run.sh
	
	Optional arguments:
		START:		  The first number to be compared (Default: 100 - Min. 1)
		END:		  The last number to be compared (Default: 1000 - Máx. 65536 * 16)
		NUM_PROCS:	  Number of computation clusters to be used (Default: 16 - Must be power of 2 - Máx. 16)
		NB_THREADS:	  Number of threads to be used in the computation cluster - OpenMP (Default: 16 - Máx. 16)
		SHOW_RESULTS: If the friendly numbers should be printed (0 = NO, 1 = YES - Default: 0)
